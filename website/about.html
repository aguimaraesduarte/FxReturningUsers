About/Methodology

Calculations are made on a representative 1% sample of main summary pings for the desktop release of Firefox. Pings are cleaned to remove subsessions longer than 24h and profile creation dates that are outside the range 2010-01-01 to the day of analysis (D0). We then select all the pings for the profiles that were active on D0. These pings are aggregated by client_id, and days of activity are saved as a list for each client_id. In this way, we obtain a table that is similar to longitudinal. By ordering the days of activity per user, we can find the number of days since last activity as well as the number of days until next activity on a per user level. We also collect the number of users who have no previous activity and/or no further activity.
